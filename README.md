_ School Management System

> Overview
The School Management System is a web based platform designed to simplify and digitize essential school administrative operations.
This project provides an organized environment for managing students, teachers, academic records, attendance, reports, announcements, and account management through an interactive web interface.
It was developed to demonstrate practical software engineering skills in backend development, database structuring, and full-stack web application design.

> Key Features

> Administration
- Manage school-wide settings
- Account management for users
- Academic year setup
- School information management

> Student Management
- Register and organize student records
- Student profile management
- Promotion of students between academic levels
- Transcript generation
- Academic performance tracking

> Teacher Management
- Teacher registration and profile handling
- Class assignment management
- Teacher reporting tools

> Academic Management
- Subject management
- Grade entry and reporting
- Attendance tracking
- Class performance reports

> Communication
- Messaging system
- School announcements
- Parent portal access

> Security
- Authentication system
- Password change and recovery system
- Role-based access structure

> Tech stack

- Python 3
- Flask
- HTML5
- CSS3
- JavaScript
- SQL Database
- Bootstrap

> Project Structure

```plaintext
app.py                # Main application
run.py                # Project launcher
config.py             # Configuration settings
backup.py             # Backup utility
cleanup.py            # Maintenance utility
schema.sql            # Database schema

/templates            # HTML templates
/static/css           # Styling
/static/js            # Frontend logic
/static/fonts         # Font assets
```

> Installation

Install dependencies:

```bash
pip install -r requirements.txt
```
Run the project:

```bash
python run.py
```
Open in browser:

```plaintext
http://127.0.0.1:5000
```
